Thimbl-CLI commands.

change
Interactively change the details of your profile. Use `reset' instead if your plan is hopelessly mangled.

fetch
Retrieve and print the messages of people that you are following.

follow NAME ADDRESS
Follow someone. E.g. thimbl follow dk dk@telekommunisten.org

help
Print some help information.

info
Print information about the current configuration of Thimbl-CLI

post MSG
Post a message. E.g. thimbl post "Hello world"

reset
!DANGER WILL ROBINSON. DATA WILL BE LOST!
Hard reset the plan file. Useful if it becomes corrupt, and you need to reset it to a default guessed state.

stdin
Post a message by reading from stdin

unfollow ADDRESS
Stop following ADDRESS. E.g. 

unfollow dk@telekommunisten.org
