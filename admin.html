This page contains notes on maintaining Thimbl-CLI.
  • Install Ruby
  • gem install grancher . This is needed to push web pages to the branch gh-pages on github. Unless your name is Mark Carter, you probably wont need to use this functionality. Forkers might want to make want to use it, too.
  • git checkout -b gh-pages origin/gh-pages. This creates a link between the local and remote repo for the website.
